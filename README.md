MyAuth APP v1.0
===============

Repo: https://github.com/mariodan/auth-app


Env
----------
- set-up your environment with:
* It's good to have NODE_ENV variable set: NODE_ENV=development|production 
* node.js >=4.2.1
* nvm - Node Version Manager
* npm - Node Package Manager
* pm2 - Process monitor
